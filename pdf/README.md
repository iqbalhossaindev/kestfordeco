# KestFord PDF Tool v2 — Real-Time PDF Editor
**Upload to: kestford.com/pdf/**

## 📁 Files
```
kestford-pdf/
├── index.html        ← Single page (landing + editor)
├── css/style.css     ← All styles (mobile + desktop)
├── js/editor.js      ← Real-time PDF canvas engine
├── js/tools.js       ← All 20+ tool panels & logic
└── README.md
```

## 🚀 Upload
Upload `kestford-pdf/` folder to your server root:
- **Landing:** `kestford.com/pdf/`
- **Editor:** Same page — click "Open Editor"

## Logo
The logo automatically loads from `../logo.png` (i.e. `kestford.com/logo.png`).
Fallback shows the "KF" badge if the image fails.

## 🛠️ Tools Included
- **Annotate:** Real-time draw, highlight, text on PDF canvas
- **Merge, Split, Compress, Convert**
- **Rotate, Reorder, Delete, Crop Pages**
- **Page Numbers, Header & Footer**
- **Fill Forms, E-Signature**
- **OCR Text Extract, AI Summary, Compare PDFs**
- **Watermark, Password Protect, Redaction, Metadata**

## 📱 Mobile + Desktop
- **Auto-detect:** Switches layout based on screen size
- **Manual toggle:** 📱/🖥️ buttons in top bar
- **Mobile:** Bottom nav + slide-up tool sheets + touch drawing
- **Desktop:** Sidebar + options panel + full canvas toolbar

## 🔑 Keyboard Shortcuts (Desktop)
- `V` = Select  `T` = Text  `D` = Draw  `H` = Highlight  `E` = Erase
- `Ctrl +/-` = Zoom in/out  `Ctrl 0` = Reset zoom
